How to Use Quick Add

  1. Step 1: Open Quick Add from your WordPress admin menu under “Products → Quick Add.”
  2. Step 2: Fill in product fields like name, price, SKU, description, categories, and image URL. Add as many product rows as you need.
  3. Step 3: Choose your product type (Simple, Variable, Downloadable, etc.) from the dropdown menu for each row.
  4. Step 4: Click “Add All Products” and watch them appear in your store—no refresh, no import files, no drama.

That’s it. You just replaced hours of product creation with a five-minute process. Quick Add is built for people who value time, control, and clean execution.

How to Install Quick Add

  3. Choose the ZIP file, upload it, and click Activate.
  4. You’ll now see a new “Quick Add” menu item under Products.

Quick Add: Frequently Asked Questions

Does Quick Add support variable or downloadable products?

Yes. You can choose Simple, Variable, Grouped, External, or Downloadable product types on a per-row basis before submitting. Downloadable products also let you specify the file name and URL directly.

Can I upload images when creating new products?

Yes. Each row includes a field for the main image URL. Just paste in a direct link (from your media library or elsewhere) and it will be applied as the featured image when the product is created.

Does this plugin create products instantly?

Yes. When you click “Add All Products,” the system runs an AJAX process in the background. There’s no page reload, and all successful products are live in your WooCommerce catalog immediately.

Is there a limit to how many products I can create at once?

There’s no hardcoded limit. Most users create 10–50 products per batch comfortably. If you’re adding 100+ at once, just give the AJAX engine a few extra seconds to process.

Will this work with my custom product fields?

Quick Add includes basic WooCommerce product fields out of the box. If you’ve registered custom fields or ACF fields, the Pro version lets you map those too.

Can I use this plugin with other product import tools?

Absolutely. Quick Add doesn’t interfere with WP All Import, native WooCommerce imports, or product syncing plugins. It’s a fast alternative—not a replacement—for when you need speed and control without file prep.
