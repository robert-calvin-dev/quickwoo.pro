How to Use Quick Edit

  1. Step 1: Go to “Products → Quick Edit” in your WordPress admin panel.
  2. Step 2: A spreadsheet-style interface will load with your current products.
  3. Step 3: Edit any fields inline — titles, prices, stock, categories, tags, etc.
  4. Step 4: Click “Save All” to push changes instantly via AJAX — no page reloads.

How to Install Quick Edit

  1. Download the ZIP file from your QuickWoo account.
  2. In WordPress, go to Plugins → Add New and click Upload Plugin.
  3. Upload the ZIP, click Activate, and you're done.
  4. Go to Products → Quick Edit to start editing.

Quick Edit: Frequently Asked Questions

What product fields can I edit?

Title, SKU, price, sale price, stock quantity, product status, visibility, tags, categories, and metadata fields. I’m still expanding what’s supported.

Does this work with variable products?

Yes, but editing variation-level data happens in a simplified form. If you're editing hundreds of variants, just note that performance depends on your hosting setup.

Can I bulk update multiple items at once?

Yes. You can make changes to as many rows as you want and then hit “Save All” to update everything via AJAX without reloading the screen.

Does it work with custom product types?

If your custom product type is built on WooCommerce's structure, yes. If it's wildly custom, it might need tweaks. If you're not sure, try the free version first.

Is this faster than WP All Import or spreadsheets?

If you're making targeted changes to live products, yes. This isn't built for mass CSV imports—it's for fast inline editing where you actually see the data.

What if I break something?

You can’t do much damage unless you're reckless with pricing or deleting content. And if something goes wrong, email me—I’ll do my best to help.
